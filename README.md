# Licence Book Management System

## 📌 Overview
The Licence Book Management System is a Java-based application developed as a 3rd semester project. It is designed to manage licence records efficiently, including user details and database operations.

## 🚀 Features
- Add new licence records
- Update existing licence details
- Delete licence records
- User registration and login system
- Database connectivity using MySQL
- Simple and user-friendly GUI

## 🛠️ Technologies Used
- Java (Swing for GUI)
- MySQL (Database)
- JDBC (Database Connectivity)
- NetBeans IDE

## 📂 Project Structure
- `src/` → Source code files
- `lib/` → External libraries (MySQL connector, JFreeChart, etc.)
- `nbproject/` → NetBeans configuration files

## ▶️ How to Run
1. Open the project in NetBeans IDE
2. Configure MySQL database using the provided `licence.sql` file
3. Add required libraries from the `lib/` folder
4. Build and run the project
